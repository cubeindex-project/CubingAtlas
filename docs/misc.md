# Miscellaneous

## Organizations

- [World Cube Association](https://www.worldcubeassociation.org/) - The World Cube Association governs competitions for mechanical puzzles commonly known as 'twisty puzzles' / [Competitions](https://www.worldcubeassociation.org/competitions) / [Records](https://www.worldcubeassociation.org/results/records) / [WCA Live](https://live.worldcubeassociation.org/) / [YouTube](https://www.youtube.com/@WorldCubeAssociationOfficial) / [GitHub](https://github.com/thewca/worldcubeassociation.org)
- [CubingUSA](https://cubingusa.org/) - CubingUSA's goal is to support and grow the US speedcubing community. We organize the US National Championships every summer
- [Speedcubing Australia](https://www.speedcubing.org.au/) - a not-for-profit, wholly volunteer-run organisation dedicated to supporting the Australian speedcubing community
- [Utah Cubing](https://utahcubing.org/) - a non-profit organization dedicated to promoting speedcubing and organizing competitions in Utah and Idaho

## Others

- [csTimer+](https://chromewebstore.google.com/detail/cstimer+/bfafldbloncllehfiaflmmjhohiobfmo) - A Chrome extension which will allow you to unlock new features like partial scrambles, multi puzzle image view and a LOT more!
- [Rubik's Ranker](https://rubiks-rank.web.app/) - Enter your average time and this tool will tell you where that time would rank you amongst WCA competitors!
- [VizCube](https://vizcube.pebor.me) - A local web app that enables you to import your solves from Twisty Timer and CSTimer and view fun statistics / [GitHub](https://github.com/Pebor/VizCube) / [Launch Announcement](https://www.speedsolving.com/threads/vizcube-fun-and-helpful-way-to-vizualize-solves.92835/)
- [Recon Viewer](https://basilio.dev/cubing/#Software) - A simple frame-by-frame video player to help out the reconstruction process
- [🔀 Scramble](https://scramble.cubing.net)
- [🔢 WCA Inspection web app](https://inspection.cubing.net)
- [🧪 Experiments (cubing.js)](https://experiments.cubing.net/cubing.js/)
- [ksolve+](https://mzrg.com/rubik/ksolve+/) - A C++ program that can find algorithms for many different puzzles
- [ksolve++](https://benwh1.github.io/web/software/ksolve++/index.html)
- [Algorithm Mirrors](https://mzrg.com/rubik/mirrors.shtml) - finds mirrors and inverses of algorithms
- [Rotation Remover](https://mzrg.com/rubik/rotations.shtml) - removes rotations from an algorithm
- [Order Calculator](https://mzrg.com/rubik/ordercalc.shtml) - finds out how many times you have to do an algorithm to get back to solved
- [3x3x3 Constraint Solver](https://mzrg.com/js/positions/) - find 3x3x3 positions that satisfy some constraints
- [Movecount Coefficient Calculator](https://trangium.github.io/MovecountCoefficient/) / [GitHub](https://github.com/trangium/trangium.github.io)
- [Cubing Pro](https://github.com/guojia99/cubing.pro) / [GitHub](https://github.com/guojia99/cubing.pro)
- [Algorithm Translator](https://cube.rider.biz/algtrans.html) - Tool for manipulating Rubik's cube algorithms
- [Scrambles Matcher](https://scrambles-matcher.worldcubeassociation.org/) - This tool enables you to assign sets of JSON scrambles generated by TNoodle to a WCIF
- [CubingTools.de](https://cubingtools.de/) - A website built for competitors and organizers who want better planning, faster insights, and less guesswork before competition day
- [CubicDB](https://cubicdb.netlify.app/) - a complete speedcubing application that gathers all kind of resources for you / [GitHub](https://github.com/isaacvr/cubicdb)
- [Bestsiteever Mosaic](https://bestsiteever.net/mosaic/) - Rubik's cube mosaic building online software  / [GitHub](https://github.com/Roman-/mosaic)
- [Cubing Contests](https://cubingcontests.com/) - a place for hosting unofficial Rubik's Cube competitions / [GitHub](https://github.com/mintydev789/RecordRanks)
- [Hagis](https://basilio.dev/cubing/#Software) - A graphical interface solver for people who want to test out algorithms / [Windows](https://basilio.dev/cubing/Hagis-v04.zip) / [OSX](https://basilio.dev/cubing/Hagis.app-v06.zip) / [Linux](https://basilio.dev/cubing/Hagis-x86_64.AppImage)
- [CubeTwister](https://www.randelshofer.ch/cubetwister/index.html) - a tool to solve your scrambled cube, to document your collection of cubes, to describe your solution algorithms, and to analyse the permutations of cube parts
