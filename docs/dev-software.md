# Cubing Ressources for Developers

- [AnimCubeJS](https://animcubejs.cubing.net/animcubejs.html) - Modern 3D JavaScript Simulator for Animating the Rubik's Cube / [GitHub](https://github.com/cubing/AnimCubeJS)
- [Cubing icons](https://icons.cubing.net/) - Icons for WCA events, unofficial events, and competition-related concepts / [GitHub](https://github.com/cubing/icons)
- [TNoodle](https://www.worldcubeassociation.org/regulations/scrambles/) - The current official scramble program / [GitHub](https://github.com/thewca/tnoodle)
- [cubing.js](https://js.cubing.net/) / [GitHub](https://github.com/cubing/cubing.js)
- [ksolve.js](https://cubing.net/ksolve.js/) - ksolve+ port to Javascript
- [ACube.js](https://cubing.net/ACube.js/) - ACube port to Javascript
- [VisualCube](https://cube.rider.biz/visualcube.php) - Generate custom Rubik's cube visualisations from your browser address bar  
  ↳ [VisualCube TS](https://github.com/tdecker91/visualcube) - Fork of VisualCube, a great Rubik's cube svg image generator
- [VisualCube Editor 2.0](https://visualcube.roudai.net/) - This is a tool for making Rubik's Cube images with GUI. VisualCube TS is used to display the images. / [GitHub](https://github.com/roudai/VisualCubeEditor2)
- [TwistySim](https://cube.rider.biz/twistysim.html) - An interactive HTML5 twisty puzzle simulator / [Demo](https://cube.rider.biz/twistysim/demo.html)
- [scramble-display](https://experiments.cubing.net/scramble-display/) - An HTML library to display puzzle scrambles / [GitHub](https://github.com/cubing/scramble-display)
- [Visual Cube Plus](https://visualcubeplus.com) - Drawing tool for regular 3x3 cubes
- [alg.cubing.net](https://alg.cubing.net/)
- [cubing.rs](https://github.com/cubing/cubing.rs) - Features from cubing.js in Rust
